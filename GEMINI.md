# Gemini CLI - Project Memory

Este projeto utiliza o **Conductor** para gerenciamento de contexto e ciclo de vida de desenvolvimento.

## Estrutura de Contexto
As definições centrais do projeto estão localizadas na pasta `conductor/`:
- [Produto](conductor/product.md): Objetivos e usuários.
- [Stack Tecnológica](conductor/tech-stack.md): Ferramentas e arquitetura.
- [Diretrizes](conductor/product-guidelines.md): Estilo e regras de negócio.
- [Workflow](conductor/workflow.md): Processo de desenvolvimento.

## Como trabalhar neste projeto
1. Sempre verifique o arquivo [tracks.md](conductor/tracks.md) para ver o progresso atual.
2. Para novas tarefas, utilize `/conductor:newTrack`.
3. Para implementar, utilize `/conductor:implement`.

---
*Nota: Siga rigorosamente as diretrizes em `conductor/product-guidelines.md`.*
