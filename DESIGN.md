# Design System Theme

## Overview
This document outlines the core visual theme for our design system, defining the fundamental aesthetic properties that govern the user interface.

## Color Palette

### Color Mode
The theme is set to `light` mode, providing a bright and airy interface experience.

### Brand Colors
- **Primary Color**: `#d4e3fe` - This soft blue serves as the main interactive color, suitable for primary actions and key elements.
- **Secondary Color**: `#afbddf` - A muted blue, used for less prominent UI elements and secondary actions.
- **Tertiary Color**: `#c6b1ec` - A gentle purple accent for highlights, badges, or decorative purposes.
- **Neutral Color**: `#95a3e8` - A mid-tone blue-grey, forming the basis for backgrounds and non-chromatic surfaces.

## Typography

### Headline Font
The font family for headlines is `inter`, chosen for its modern and legible appearance.

### Body Font
The font family for body text is `inter`, ensuring consistent readability across all textual content.

### Label Font
The font family for labels is `inter`, maintaining typographic harmony throughout interactive elements.

## Shape and Form

### Roundedness
UI elements feature a `moderate` level of roundedness (value: `2`), striking a balance between sharpness and softness for a friendly yet structured feel.

## Spacing

### Spacing Scale
The theme employs a `normal` level of spacing (value: `2`), providing adequate whitespace for visual clarity and comfortable user interaction.
